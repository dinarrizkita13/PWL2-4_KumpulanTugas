Cara membuat folder laravel setelah mendownload composer,xampp,dan php versi 8.2+

Buka folder yang mau di pakai,buka cmd di dalam folder tersebut.
kemudian paste kan command ini "composer create-project laravel/laravel nama_proyek_anda".
contoh nya "composer create-project laravel/laravel myproject",'myproject' itu nama folder yang akan digunakan di laravel.
